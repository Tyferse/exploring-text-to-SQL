## Текушие наработки

Для оценки используется бенчмарк Spider 2.0 (преимущественно датасет Spider 2.0-lite).

На данный момент проведён анализ затрат и ошибок двух baseline методов: (DIN-SQL)[] и (DAIL-SQL)[] и одного из топа по бенчмарку - (ReFoRCE)[]. Результаты располагаются в папке `analysis`.

Планируется проанализировать метод (AutoLink)[], также из топа Spider 2.0-lite.
